# Консультации Ольги Розет

**Сайт:** `olgarozet.ru` (планируется)  
**Статус:** MVP (разработка)  
**Дата создания:** 06.11.2025

---

## Описание

Публичная услуга: 50-минутные консультации для практикующих дизайнеров интерьеров.

**Философия:** Формализация существующей практики супервизии выпускников ВБШД. "Защита времени" вместо монетизации.

**Tone of voice:** "Не продающий", доверительный, экспертный (конгруэнтность Paris 2026).

---

## Архитектура

**Принцип:** MHE (Минимальное человеческое усилие)

**Технический стек:**
- Статический HTML/CSS (минимализм)
- GitHub Pages (хостинг)
- Calendly (бронирование, inline виджет)
- Самозанятость (НПД) + прямой банковский перевод (оплата)

**Референс стиля:** `parisinjanuary.ru`

---

## Структура страницы

1. **Hero** — Имя, функция ("50 минут: валидация решений")
2. **Философия** — "От неформальных звонков к защите времени"
3. **Для кого** — Фильтр аудитории (mid-level дизайнеры)
4. **Формат и Оплата** — 50 минут, цена, НПД
5. **Куратор** — Био Ольги, фото
6. **Запись** — Calendly виджет
7. **Footer** — Email, Instagram

---

## Процесс бронирования (MVP)

1. Клиент выбирает слот в Calendly
2. Calendly собирает: Имя, Email, Время
3. Автоматическое письмо клиенту: реквизиты для перевода
4. Клиент переводит → присылает подтверждение
5. Ольга проверяет → чек в "Мой налог" → подтверждение сессии

**Масштабирование (>10 клиентов/мес):** Интеграция ЮKassa (автоматизация)
