# Настройка DNS для olgaroset.ru (Рег.ру → GitHub Pages)

**Дата:** 06.11.2025  
**Домен:** olgaroset.ru (зарегистрирован на Рег.ру)  
**Цель:** Направить домен на GitHub Pages

---

## Шаг 1: Создать GitHub репозиторий

**Я сделаю:**
1. Создам репозиторий `olgaroset` на твоём GitHub
2. Загружу туда файлы (index.html, style.css)
3. Настрою GitHub Pages

**После этого GitHub даст адрес:** `azrozet.github.io/olgaroset`

---

## Шаг 2: Настроить DNS на Рег.ру

**Тебе нужно (в личном кабинете Рег.ру):**

### Вариант A: CNAME (рекомендуется)

1. Зайти в управление доменом `olgaroset.ru`
2. Раздел "DNS-серверы и зона"
3. Добавить запись:

```
Тип: CNAME
Имя: @
Значение: azrozet.github.io
TTL: 3600
```

4. Добавить вторую запись (для www):

```
Тип: CNAME
Имя: www
Значение: azrozet.github.io
TTL: 3600
```

---

### Вариант B: A-записи (альтернатива)

Если CNAME не работает, используй A-записи:

```
Тип: A
Имя: @
Значение: 185.199.108.153
TTL: 3600
```

Добавить ещё 3 A-записи с теми же параметрами, но другими IP:

```
185.199.109.153
185.199.110.153
185.199.111.153
```

(Это IP-адреса GitHub Pages)

---

## Шаг 3: Настроить CNAME в GitHub

**Я сделаю:**
1. Создам файл `CNAME` в репозитории
2. Пропишу в нём: `olgaroset.ru`
3. GitHub поймёт, что домен привязан

---

## Шаг 4: Подождать

**DNS обновление:** 5-30 минут (иногда до 24 часов)

**После этого:**
- `olgaroset.ru` → откроется сайт
- `www.olgaroset.ru` → откроется сайт
